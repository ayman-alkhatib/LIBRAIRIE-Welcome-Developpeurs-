Package pour générer les PV au format .docx (et éventuellement odt avec modification)
utilise la librairie TinyButStrong

charger le dossier pv/ sur le serveur distant

créer un sous-domaine éventuellement pointant vers pv/

page de génération de PV : pv/php/index.php

page de log : pv/php/log.php

liste des fichiers générés : pv/export/

liste des fichiers 'modèles' appelés templates : pv/template/


création d'un nouveau template :
créer un nouveau fichier .docx, le copier dans php/template/
dans php/index.php : ajouter un bouton "générer XXX.docx"
dans php/template-save.php : vers la ligne 790, faire un nouveau cas concernant ce nouveau modèle


ajout d'un nouveau Bureau / GE / Technicien :
mettre-à-jour GE vers la ligne 191 de php/index.php
mettre-à-jour dans les fichiers :
php/func/chg_bureau.php
php/func/chg_dept.php
php/func/chg_tech.php
